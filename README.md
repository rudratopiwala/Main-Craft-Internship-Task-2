# Main-Craft-Internship-Task-2

## Titanic Survival Analysis

## Project Overview

This project analyzes the famous **Titanic dataset** to understand survival patterns of passengers.

The analysis includes data cleaning, exploration, and visualization using Python.

---

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Dataset

* Dataset: `titanic.csv`
* Contains passenger details such as age, gender, class, and survival status.

---

## Tasks Performed

* Data loading using Pandas
* Handling missing values (Age, Embarked)
* Dropping unnecessary columns (Cabin)
* Exploratory Data Analysis (EDA)
* Survival pattern analysis

---

## Analysis Questions

* Who survived more: males or females?
* Did passenger class affect survival chances?
* What was the survival rate by age group?

---

## Visualizations

* Bar chart of survival by gender
* Bar chart of survival by passenger class
* Histogram of passenger ages

---

## Key Insights

* Females had a higher survival rate than males.
* First-class passengers had better survival chances.
* Younger passengers had relatively higher survival rates.

---

## How to Run

1. Clone the repository
2. Open Jupyter Notebook
3. Run `titanic_analysis.ipynb`
